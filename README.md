# Mobile-Apps-EOY-Project

#Application Name: HWMaster: Homework Manager
#Team Members: Vishakh Hari, Abhinav Tanniru
Description:  HWMaster is a homework management app designed to help students stay organized with minimal possible effort. The app allows users to quickly add assignments sorted by class, with each subject color coded for easy tracking and prioritizing. Assignments can be marked when complete with a check off feature, and the app tracks the user's prgress. The goal of our app is to reduce the stress of managing multiple assignments and mitigating the common occurence of students forgetting about assignments, by offering an user-friendly interface that can keep track for them.

From an architectural standpoing, the application follows a simple yet effective structure, separating the user interface form data handling to ensure smooth performance and scalability. The front end presents assignments in an organized list format, while the underlying logic manages task creation, categorization, and completion status. This design ensures that the app remains lightweight while still delivering a polished and responsive user experience.
